# next-data-fetching

Next.js App Router data fetching decision tree focused on TTFB-first and honest UX.

## Install (skills.sh)

```bash
npx skills add BartdBont/next-data-fetching
```

## What You Get

- `SKILL.md`: the workflow and rules of thumb
- `references/decision-tree.md`: the full decision tree (with Mermaid)
- `references/patterns-app-router.md`: implementation templates (App Router)
- `references/examples.md`: worked examples
- `references/review-checklist.md`: code review checklist
